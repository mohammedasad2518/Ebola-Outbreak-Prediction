# 📌 Overview

Ebola Outbreak Prediction & Analytics System is a Machine Learning and Data Analytics project that analyzes historical Ebola outbreak data to predict future cases and deaths using predictive modeling techniques.

The project combines:
- 📊 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning Models
- 📈 Future Forecasting
- 🌍 Interactive World Map Visualization
- 📉 Mortality & Growth Rate Analysis
- 📋 Professional Dashboards & Visualizations

This project demonstrates real-world healthcare analytics and epidemic forecasting using Python and Machine Learning.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/mohammedasad2518/Ebola-Outbreak-Prediction.git
```

Go to the project directory:

```bash
cd Ebola-Outbreak-Prediction
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Jupyter Notebook:

```bash
jupyter notebook
```
